# Vevhu Field Mobile — Modern Premium Design System (3 Variants)

## Variant 1: Vevhu Corporate Signature (Default Brand)
- **Primary Color**: `#F3772D` (Warm Vevhu Orange)
- **Secondary Accent**: `#1976D2` (Vevhu Corporate Blue)
- **Background**: Light `#FAFAF9` / Slate `#0A0E0F`
- **Cards**: Surface `#FFFFFF` with `#E7E5E4` subtle border and `elevation: 4`
- **Typography**: Bold Sans-Serif (24px H1, 16px H3, 14px Body)
- **Highlights**: Glassmorphism action cards, dynamic safe-area bottom navigation, high-contrast stat boxes.

## Variant 2: Midnight Tactical Field (High-Sunlight Dark Mode)
- **Primary Color**: `#10B981` (Field Emerald) + `#F59E0B` (Tactical Amber)
- **Background**: OLED Deep Black `#090D0E`
- **Cards**: `#161B22` with `#30363D` border
- **Typography**: High-contrast monospaced data indicators (26px Stat Counters)
- **Highlights**: Anti-glare outdoors optimization, neon status indicators, tactile physical touch feedback.

## Variant 3: Minimalist Enterprise Clean (Executive Light)
- **Primary Color**: `#1D4ED8` (Royal Sapphire Blue) + `#F97316` (Coral Accent)
- **Background**: Clean Pure `#FFFFFF`
- **Cards**: `#F8FAFC` with `#E2E8F0` soft borders
- **Typography**: Clean Helvetica/Inter hierarchy with extra whitespace
- **Highlights**: Floating pill segment controls (`👤 My Submissions` vs `🏢 All Team Submissions`), minimal borders.
